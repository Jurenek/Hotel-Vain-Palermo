# Logo del Hotel VAIN

## Cómo agregar el logo:

1. **Guarda el logo del hotel** en esta carpeta:
   - Ruta: `public/assets/`
   - Nombre sugerido: `vain-logo.png` o `vain-logo.svg`

2. **El logo se mostrará en:**
   - Welcome screen (página principal)
   - Puede usarse en otras páginas si quieres

## Formato recomendado:
- **PNG** con fondo transparente
- **SVG** (vector - mejor calidad)
- Tamaño mínimo: 400x400px

## Una vez que subas el logo:
Decime y actualizo la app para usar tu logo real en vez de la foto genérica.
